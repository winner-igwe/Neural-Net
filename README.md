# Modular Neural Network from Scratch

## Objective
A modular implementation of a foundational feedforward neural network (Multi-Layer Perceptron), built and trained from scratch. 

Instead of relying on high-level library abstractions, this project was built to master the underlying calculus and linear algebra of deep learning. It focuses on the fundamental mechanics of artificial intelligence, specifically forward propagation, manual backpropagation, and gradient descent optimization.

## Technical Stack
* **Language:** Python
* **Core Focus:** Algorithmic infrastructure, linear algebra, and supervised learning.

## Development Roadmap
While the current architecture successfully implements a modular MLP, the future scope for this repository includes:
* Extending the modular framework to support Convolutional Neural Networks (CNNs) for spatial data processing.
* Implementing Recurrent Neural Networks (RNNs) for time-series forecasting (with applications intended for battery degradation analysis).

## Context
This from-scratch implementation served as the foundational stepping stone before transitioning into PyTorch-based Scientific Machine Learning and Physics-Informed Neural Networks (PINNs).
